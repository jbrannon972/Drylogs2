# Drylogs Install Workflow - User Guide

## Tech User: Complete an Install

### 1. Login
1. Open Drylogs app
2. Enter email and password
3. Click **Sign In**

### 2. Start Install
1. Find a job in **Preinstall** phase
2. Click **Start Install**

### 3. Workflow Steps

**Step 1: Office Prep (Optional)**
- Review job details
- Click Continue

**Step 2: Arrival**
- Enter arrival time
- Take photo of front door
- Click Continue

**Step 3: Customer Introduction**
- Complete checklist (intro, walkthrough, utilities)
- Mark any concerns
- Click Continue

**Step 4: Cause of Loss**
- Select water source type and category
- Take photos of damage
- Click Continue

**Step 5: Room Assessment**
- For each room:
  - **Basic Info:** Enter name, dimensions, affected areas
  - **Moisture:** Add readings and photos
  - **Materials:** Check boxes for materials to remove
  - **Pre-existing:** Document any prior damage
  - **Photos:** Take 4+ room photos
- Save room and repeat for all rooms

**Step 6: Drying Chambers**
- Create chambers (e.g., "Chamber 1")
- Assign rooms to each chamber
- Save chambers

**Step 7: Partial Demo (Optional)**
- If doing demo work, log materials removed
- Take demo photos
- Skip if no demo

**Step 8: Equipment**
- Review calculated equipment needs
- (Optional) Scan and place equipment in rooms

**Step 9: Billable Work (Optional)**
- Add any extra services performed
- Skip if none

**Step 10: Final Photos**
- Take 4+ photos of equipment setup
- Click Continue

**Step 11: Schedule**
- Add future monitoring visits
- Set dates and visit types
- Save schedule

**Step 12: Communicate Plan**
- Review on-screen plan with customer
- Click Continue

**Step 13: Complete**
- Enter departure time
- Add tech notes (optional)
- Click **Complete Install Workflow**

### 4. After Completion
- Job moves to next phase
- Return to Tech Dashboard

## PSM User: View Initial Report

### 1. Login as PSM
1. Open Drylogs app
2. Login with PSM credentials

### 2. Access PSM Dashboard
1. Click **PSM Dashboard** in navigation

### 3. Generate Report
1. Find the completed job in the job list
2. Look for the **"Generate Initial Inspection Report"** button
   - Button only appears after install is completed
3. Click the button
4. Report opens in new window with logo, room details, moisture readings, materials to remove, and equipment
